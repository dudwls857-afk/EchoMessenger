# (C# 코딩) 에코메신저

## 개요
- C# 프로그래밍 학습
- 1줄소개: 사용자 키보드 입력을 받아 메시지를 출력하고 관리하는 에코 메신저 프로그램
- 사용한 플랫폼:
  - C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
  - Label, TextBox, ListBox, Button

## 사용한 기술과 구현한 기능
- Visual Studio를 이용하여 Windows Forms 기반 UI 설계
- TextBox를 통해 사용자 입력 데이터를 받아 처리
- Button 클릭 이벤트를 이용하여 입력된 메시지를 ListBox에 출력
- ListBox의 Items.Add()를 활용하여 메시지 목록 관리
- TextBox.Clear()를 사용하여 메시지 전송 후 입력창 초기화
- string.IsNullOrWhiteSpace()를 활용하여 공백 입력 방지
- DateTime.Now를 이용하여 현재 시간 정보를 메시지에 추가
- string.Trim()을 사용하여 입력값의 앞뒤 공백 제거
- ListBox.Items.Count를 활용하여 메시지 개수 실시간 표시
- ListBox.SelectedIndex를 이용하여 선택된 메시지 삭제 기능 구현
- ListBox.Items.Clear()를 통해 전체 대화 내용 초기화 기능 구현
- 입력 글자 수를 제한하여 사용자 입력 검증 기능 구현

- 과제 내용
- Label, TextBox, Button, ListBox를 적절히 배치하여 기본적인 UI를 구성하였다.
- TextBox에 입력한 메시지를 전송 버튼 클릭 시 ListBox에 추가되도록 구현하였다.
- 메시지가 추가된 후에는 TextBox를 Clear() 메소드를 사용하여 자동으로 초기화되도록 설정하였다.

- 입력된 메시지는 한 줄씩 계속 누적되어 이전에 입력한 내용도 확인할 수 있도록 하였다.
- 메시지 전송 이후에는 입력창이 자동으로 비워져 다음 메시지를 편리하게 입력할 수 있도록 구성하였다.

- 과제 내용

- 전송 버튼 클릭 시 TextBox의 텍스트를 ListBox의 항목(Items)으로 추가합니다.

- 구현 내용과 기능 설명
- 사용자가 입력창에 메시지를 입력한 후 전송 버튼을 클릭하면 리스트 박스에 메시지가 추가되도록 구현하였다. 
- 메시지 전송 이후 입력창이 자동으로 초기화되고 커서가 다시 입력창으로 이동하도록 하여 연속 입력이 가능하도록 하였다.
- Enter 키를 눌러도 전송이 이루어지도록 설정하였다.
- 공백이나 빈 문자열이 입력될 경우 메시지가 전송되지 않도록 조건문을 통해 입력을 제한하였다.

- 과제 내용
- Label, TextBox, Button, ListBox를 적절히 배치합니다.
- 전송 버튼 클릭 시 TextBox의 텍스트를 ListBox의 항목(Items)으로 추가합니다.

- 구현 내용과 기능 설명
- 사용자가 리스트 박스에서 특정 메시지를 선택한 후 삭제 버튼을 누르면 해당 항목이 제거되도록 구현하였다.

- 메시지 입력 시 글자 수를 50자로 제한하여 초과할 경우 경고 메시지를 출력하고 전송되지 않도록 구현하였다.
